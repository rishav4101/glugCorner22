GLUG Corner

Founded in 2003, The GNU/Linux Users' Group, National Institute Of Technology, Durgapur is the official open-source club of this institution. At the precipice of bleeding edge and the free software movement, we reside to spread the culture of collaboration, free software and togetherness among our institute's students.

DISTRO

UBUNTU
Ubuntu is a Linux distribution based on Debian and composed mostly of free and open-source software.
GITHUB
ARCH LINUX
Arch Linux is a Linux distribution created for computers with x86-64 processors. Arch Linux adheres to the KISS principle.
GITHUB
FEDORA
Fedora Linux is a Linux distribution developed by the Fedora Project which is sponsored primarily by Red Hat with additional support and sponsors.
GITHUB
MANJARO
Manjaro is a free and open-source Linux distribution based on the Arch Linux operating system. Manjaro has a focus on user-friendliness and accessibility.
GITHUB

BROWSER

FALKON
Falkon is a free and open-source web browser. It is built on the Qt WebEngine which is a wrapper for the Chromium browser core.
GITHUB
CHROMIUM
Chromium is a free and open-source web browser project, principally developed and maintained by Google.
GITHUB
FIREFOX
Mozilla Firefox is a free and open-source web browser that uses the Gecko rendering engine to display web pages.
GITHUB

OFFICE

LIBRE OFFICE
LibreOffice is a free and powerful office suite. Its clean interface and feature-rich tools help you unleash your creativity and enhance your productivity.
GITHUB
SEJDA
Sejda is an open source, task oriented Java library to edit PDF files.
GITHUB

GRAPHICS

BLENDER
Blender is a free and open-source 3D computer graphics software toolset used for creating animated films, motion graphics, 3D applications etc.
GITHUB
KDENLIVE
Kdenlive is a free and open-source video editing software based on the MLT Framework, KDE and Qt.
GITHUB
SHOTCUT
Shotcut is a free and open-source, cross-platform video editing app.
GITHUB

MEDIA

VLC MEDIA PLAYER
VLC media player is a free and open-source, portable, cross-platform media player software and streaming media server.
GITHUB
AUDACITY
Audacity is a free and open-source digital audio editor and recording application software.
GITHUB
INKSCAPE
Inkscape is a free and open-source vector graphics editor used to create vector images, primarily in Scalable Vector Graphics format.
GITHUB
GIMP
GIMP is a free and open-source raster graphics editor used for image editing, transcoding between different image file formats, and more specialized tasks.
GITHUB

DEV

ATOM
Atom is a free and open-source text and source code editor with embedded Git Control. Developed by GitHub, Atom is a desktop application built using web technologies.
GITHUB
GODOT ENGINE
Godot is a 2D and 3D, cross-platform, free and open-source game engine. The development environment runs on multiple operating systems including Linux, macOS, and Windows.
GITHUB
QEMU
QEMU is a free and open-source emulator. It emulates the machine's processor and provides a set of different hardware and device models, enabling it to run guest operating systems.
GITHUB
VIM
Vim is a free and open-source, screen-based text editor program for Unix. It is an improved clone of Bill Joy's vi. It is also highly configurable.
